# SOC Dashboard (Python)

## Features
- Real-time process monitoring
- Risk classification (HIGH / MEDIUM / LOW)
- Flask dashboard with filtering
- SQLite storage

## Requirements
- Windows OS (uses WMI)

## Setup
pip install -r requirements.txt

## Run
Terminal 1:
python app.py

Terminal 2:
python monitor.py

Open:
http://127.0.0.1:5000
